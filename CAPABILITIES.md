## How to Access These Analyses

### Via Claude Code Terminal:
```bash
python3 << 'EOF'
from pycostaudit import PyCostAudit

auditor = PyCostAudit()

# Example: Run anomaly detection
anomalies = auditor.detect_anomalies()
print(anomalies)

# Example: Cost by project
costs_by_project = auditor.analyze_by_project()
print(costs_by_project)
EOF
```

### Via CLI Commands:
```bash
# Setup daily reports
cost-audit setup --daily-email your@email.com

# Set budget alert
cost-audit alert --monthly 500

# Generate report
cost-audit report --format pdf --period monthly
```

### Via Web Dashboard:
```bash
cost-audit dashboard
# Opens interactive dashboard at http://localhost:8080
```
